# 翻譯術語表 / 譯者須知

## 給譯者（Opus 4.6）的規則

逐檔翻譯 `jp/` 下的 JSON：**只填每個 entry 的 `zh` 欄**，其餘欄位（id / line / type / jp）一律不要改。

標籤規則：
- `[...]` 中括號是引擎指令，**原樣保留**。最常見的 `[p]`（換頁）必須留在句尾。
- `[srb text="ふたば"]双葉[erb]` 是日文振假名注音：翻成中文時**整段換成譯詞**即可（例：`雙葉`），不需要保留 srb/erb。
- `「」` 對白引號保留。`type=ptext` 的譯文保留 `-&nbsp; ... &nbsp;-` 這類結構，只換中間的日文。
- `type=name` 的 `jp` 不含開頭的 `#`，直接給譯名即可。

- 統一使用**繁體中文**。空 `zh` 會在回填時保留日文原文，可分批翻、分批測試。
